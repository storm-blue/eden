# 星星城背景音乐

## 文件说明

请将星星城的背景音乐文件放置在此目录下：

- `star-city-bg.mp3` - MP3 格式的背景音乐
- `star-city-bg.ogg` - OGG 格式的背景音乐（可选，用于更好的浏览器兼容性）

## 音乐要求

- 建议使用轻柔、梦幻的背景音乐
- 音乐应该能够无缝循环播放
- 文件大小建议控制在 2MB 以内
- 音量适中，不要过于突兀

## 使用方式

音乐会在以下情况自动播放/停止：
- 进入星星城页面时：自动播放
- 关闭星星城页面时：自动停止

## 音频格式支持

为了确保最佳的浏览器兼容性，建议同时提供 MP3 和 OGG 两种格式。
